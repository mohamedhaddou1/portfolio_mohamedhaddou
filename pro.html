Analyse et Reporting des Ventes

Période : Novembre à Décembre 2025

Objectif : analyser les données de ventes et automatiser le reporting décisionnel.

Technologies utilisées : Python (pandas, Google Colab), Power Query (Script M), Power BI.

Compétences acquises :

  • Exploration, nettoyage et transformation de données avec Python & pandas
  • Calcul de métriques clés (chiffre d’affaires, quantités vendues, best-sellers)
  • Visualisation des ventes pour identifier les tendances
  • Création d’un pipeline de données automatisé avec Power Query
  • Développement de dashboards interactifs sous Power BI avec actualisation automatique
