Projets scolaires & personnels

GLPI - Gestion de tickets

Objectif : centraliser et suivre les incidents.

Rôles : Employé (création et suivi), Administrateur (gestion des employés et traitement des tickets).

Technologies utilisées : PHP, MySQL, HTML, CSS, JavaScript, Merise.

Compétences acquises :

  • Conception de bases de données relationnelles
  • Gestion des rôles et des workflows
  • Création d’un tableau de bord dynamique

Jeu de l'aventurier - Jeu vidéo

Objectif : développer un jeu vidéo basé sur une architecture MVC (Modèle - Vue - Contrôleur) permettant de gérer un personnage, ses déplacements, ses combats et son inventaire.

Technologies utilisées : Java (Eclipse IDE).

Compétences acquises :

  • Application du modèle MVC pour organiser le code
  • Gestion d’entités du jeu (joueur, ennemis, objets, inventaire, carte)
  • Mise en place de contrôleurs pour gérer les actions (combat, déplacement, inventaire)
  • Création de vues textuelles pour l’affichage interactif
  • Manipulation de classes, objets et packages en Java

Menu interactif d'un jeu vidéo

Objectif : créer des questions qui interroge l'utilisateur pour concevoir l'identité du personnage, et élaborer un menu interractif lui permettant de gérer le comportement, ses déplacements et son inventaire.

Technologies utilisées : Java (Eclipse IDE).

Compétences acquises :

  • Gestion des entrées utilisateur (console / scanner)
  • Utilisation des structures conditionnelles et boucles
  • Conception d’une interface textuelle simple et interactive
  • Organisation du code en méthodes pour plus de clarté
