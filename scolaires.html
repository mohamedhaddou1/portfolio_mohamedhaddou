Projets scolaires & personnels

Resa_VVA - Application de réservation (Client léger)

Objectif : créer une application web de réservation d’hébergements.

Fonctionnalités :

  • Client : consulter les hébergements
  • Réserver par semaine
  • Choisir un créneau de nettoyage
  • Gestionnaire : ajouter, modifier et supprimer des hébergements
  • Gérer les réservations et consulter le planning

Technologies utilisées : HTML, CSS, JavaScript, PHP, MySQL (XAMPP).

Compétences acquises :

  • Développement web
  • Frontend / Backend
  • Base de données
  • Système de réservation

Proxibanque - Application bancaire (Client lourd)

Objectif : développer une application de gestion de comptes bancaires pour différents utilisateurs.

Architecture : application en architecture multicouche (Controller, Service, DAO, Model, Vue).

Fonctionnalités :

  • Client : consulter ses comptes, voir les soldes, effectuer des virements
  • Faire des demandes (carte, chéquier, prêt)
  • Modifier ses informations et consulter ses statistiques
  • Conseiller : consulter les comptes clients
  • Gérer les demandes et consulter les statistiques

Technologies utilisées : Java, JavaFX, FXML, CSS, MySQL (XAMPP).

Compétences acquises :

  • Développement client lourd
  • Architecture multicouche
  • Connexion base de données
  • Gestion des rôles

Jeu de l'aventurier - Jeu vidéo

Objectif : développer un jeu vidéo basé sur une architecture MVC (Modèle - Vue - Contrôleur) permettant de gérer un personnage, ses déplacements, ses combats et son inventaire.

Technologies utilisées : Java (Eclipse IDE).

Compétences acquises :

  • Application du modèle MVC pour organiser le code
  • Gestion d’entités du jeu (joueur, ennemis, objets, inventaire, carte)
  • Mise en place de contrôleurs pour gérer les actions (combat, déplacement, inventaire)
  • Création de vues textuelles pour l’affichage interactif
  • Manipulation de classes, objets et packages en Java
